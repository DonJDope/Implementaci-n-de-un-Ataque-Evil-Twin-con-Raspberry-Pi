# Implementacion-de-un-Ataque-Evil-Twin-con-Raspberry-Pi
Proyecto educativo que demuestra cómo funciona un ataque Evil Twin usando una Raspberry Pi. Crea un punto de acceso falso, muestra un portal cautivo y captura credenciales simuladas para concientizar sobre riesgos en redes WiFi públicas. Desarrollado solo en un entorno controlado.
